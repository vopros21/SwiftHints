#### Hotkeys to navigate (select, perform changes) in XCode.
###### Editor view
- Fold current block: ⌥ + ⌘ + ← / →
- Fold methods and functions blocks: ⇧ + ⌥ + ⌘ + ← / →
- Fold comment blocks: ⇧ + ⌃ + ⌘ + ← / →

###### Selection
- Select word: ⇧ + ⌥ + ← / →
- Select element (word/operand/variable): ⇧ + ⌃ + ← / →

###### Edit
- Formatt to multiple lines: ⌃ + m
- Re-indent: ⌃ + i

###### Search
- Show quick search: ⇧ + ⌘ + O
