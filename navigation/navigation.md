#### Hotkeys to navigate (select, perform changes) in XCode.
###### Editor view
- Collapse/expand current block: ⌥ + ⌘ + ← / →
- Collapse/expand all blocks: ⇧ + ⌥ + ⌘ + ← / →

###### Selection
- Select word: ⇧ + ⌥ + ← / →
- Select element (word/operand/variable): ⇧ + ⌃ + ← / →

###### Edit
- Place inline arguments in separated lines: ⌃ + m

###### Search
- Show quick search: ⇧ + ⌘ + O
